# 🛒 E-commerce Sales Dashboard (Power BI)

This project showcases an interactive and insightful **E-commerce Sales Dashboard** built using **Power BI**. It enables businesses to monitor key sales metrics, identify top-performing products, and make data-driven decisions.

---

## 📊 Dashboard Overview

The Power BI dashboard includes:

- 📦 **Total Sales**, **Total Orders**, and **Profit**
- 🌍 **Sales by Country/Region**
- 🛍️ **Top Products by Revenue**
- 📈 **Monthly Sales Trend**
- 👤 **Customer Segmentation**
- 💼 **Category-wise Performance**

---

## 🛠 Tools & Technologies Used

- **Power BI Desktop**
- **Microsoft Excel** (for data cleaning)
- **DAX** (Data Analysis Expressions)
- **Power Query** (ETL)

---

## 📁 Dataset

The dataset used includes:

- Order ID, Product Name, Category, Sub-Category
- Order Date, Ship Date, Region, Country
- Sales, Quantity, Discount, Profit, and Customer Details
